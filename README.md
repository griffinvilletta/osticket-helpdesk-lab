# osTicket Help Desk Lab

This project simulates a real help desk ticketing system using osTicket and HeidiSQL 

Tools: 
-Virtual box
-Windows Server 2025
-Windows 11 Pro
-ADDS, DNS, DHCP, and IIS
-PHP
-MySQL and HeidiSQL
